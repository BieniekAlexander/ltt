# Setup
## Preconditions
- Backend running

## Supply Environment Variables
```
cat > .env <<EOF
REACT_APP_BACKEND_URL=localhost:5000
EOF
```


## Running
```
npm start	# runs app in development mode
npm test
npm run build
npm run reject
```
